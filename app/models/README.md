# 💾 Models Module

Database models and data structures for the Biometric Service.

## Overview

This module defines MongoDB document models using Pydantic for validation and serialization. Models represent core entities like face records, embeddings, and metadata.

## 📁 Structure

```
models/
├── __init__.py              # Module initialization
├── face.py                 # Face document model
├── embedding.py            # Face embedding model
├── verification_log.py     # Verification audit log
├── merchant_face.py        # Merchant face linking
└── base.py                 # Base model classes
```

## 📋 Core Models

### Face Model
```python
from models.face import Face

class Face(BaseModel):
    face_id: str                    # Unique identifier
    user_id: str                    # Associated user
    embedding: List[float]          # Face embedding vector
    image_url: str                  # S3 URL to face image
    label: str = "primary"          # Face label
    created_at: datetime
    updated_at: datetime
    is_active: bool = True
    confidence: float               # Initial confidence score
```

**Database:** Stored in MongoDB `faces` collection

### Embedding Model
```python
class FaceEmbedding(BaseModel):
    embedding_id: str               # Unique ID
    face_id: str                    # Associated face
    vector: List[float]             # 128-dimensional vector
    model_version: str              # Model used for embedding
    created_at: datetime
```

**Database:** Stored in MongoDB `embeddings` collection

### Verification Log Model
```python
class VerificationLog(BaseModel):
    log_id: str
    face_id: str
    query_embedding: List[float]
    match: bool
    confidence: float
    distance: float
    verified_by: str                # Service/user
    verified_at: datetime
    error: Optional[str] = None
```

**Database:** Stored in MongoDB `verification_logs` collection for audit trail

### Merchant Face Model
```python
class MerchantFace(BaseModel):
    merchant_face_id: str
```

**Database:** Stored in MongoDB `merchant_faces` collection

## 🏗️ Base Model

```python
from models.base import BaseModel, TimestampMixin

class TimestampMixin(BaseModel):
    """Adds created_at and updated_at timestamps"""
    created_at: datetime = Field(default_factory=datetime.utcnow)
    updated_at: datetime = Field(default_factory=datetime.utcnow)

class BaseModel(TimestampMixin):
    """Base model for all documents"""
    id: str = Field(default_factory=lambda: str(uuid.uuid4()))
    is_deleted: bool = False
```

## 📊 Field Descriptions

### face_id (String)
- Unique identifier for face record
- Format: `face_<uuid>`
- Auto-generated if not provided

### user_id (String)
- User identifier from Payment Service
- Links face to user account
- Indexed for fast queries

### embedding (List[float])
```python
# 128-dimensional vector
# Generated by face recognition model
# Similarity: Euclidean distance or cosine similarity
# Range: -1 to 1 (normalized)
```

### image_url (String)
- AWS S3 URL to original face image
- Format: `s3://bucket/faces/{face_id}/{timestamp}.jpg`
- Public read-only access

### confidence (Float)
```python
# 0.0 - 1.0
# How confident model is about face detection
# Threshold: Usually 0.6+
```

### label (String)
```python
# "primary" - main face
# "secondary" - backup face
# "merchant" - merchant face
# "test" - test/sample face
```

## 🧪 Model Usage Examples

### Creating a Face
```python
from models.face import Face
from datetime import datetime

face = Face(
    face_id="face_abc123",
    user_id="user_456",
    embedding=[...],  # 128 float values
    image_url="s3://bucket/faces/face_abc123/2025-11-12.jpg",
    label="primary",
    confidence=0.95
)

# Save to MongoDB
db.faces.insert_one(face.dict())
```

### Retrieving a Face
```python
face_data = db.faces.find_one({"face_id": "face_abc123"})
face = Face(**face_data)
print(face.user_id)
```

### Updating a Face
```python
face = Face(**existing_data)
face.updated_at = datetime.utcnow()
face.confidence = 0.98

db.faces.update_one(
    {"face_id": face.face_id},
    {"$set": face.dict(exclude=["id"])}
)
```

### Querying Faces
```python
# Find all faces for user
faces = db.faces.find({"user_id": "user_456"})
face_objects = [Face(**doc) for doc in faces]

# Find active faces
active_faces = db.faces.find({
    "user_id": "user_456",
    "is_active": True
})

# Find recent verifications
recent_logs = db.verification_logs.find({
    "verified_at": {"$gte": datetime.utcnow() - timedelta(hours=24)}
})
```


## 📝 Adding New Models

1. **Create file** in `models/` directory
2. **Define Pydantic class** inheriting from `TimestampMixin`
3. **Add validators** for complex fields
4. **Document** with docstrings
5. **Add to** `__init__.py`
6. **Create** MongoDB collection and indexes

Example:
```python
# models/custom_model.py
from models.base import TimestampMixin

class CustomModel(TimestampMixin):
    """Model description"""
    custom_field: str
    
    @field_validator('custom_field')
    def validate_custom_field(cls, v):
        if not v:
            raise ValueError('Field cannot be empty')
        return v
```
